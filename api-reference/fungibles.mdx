# Fungibles

The Fungibles endpoints provide access to fungible token information including prices, metadata, and historical charts.

## What you can do

<CardGroup cols={2}>
  <Card title="Token Search" icon="magnifying-glass">
    Search for tokens by name, symbol, or contract address.
  </Card>
  <Card title="Token Details" icon="circle-info">
    Get comprehensive metadata including links, descriptions, and market data.
  </Card>
  <Card title="Price Charts" icon="chart-line">
    Retrieve historical price data for any time period.
  </Card>
  <Card title="Market Data" icon="chart-candlestick">
    Access current prices, market cap, volume, and more.
  </Card>
</CardGroup>

## Key Features

- **Cross-chain data** — Token information across all supported chains
- **Implementation details** — Contract addresses on each chain
- **Rich metadata** — Icons, descriptions, social links
- **Historical prices** — Chart data for various time periods

## Common Parameters

| Parameter | Description |
|-----------|-------------|
| `fungible_id` | Unique identifier for the fungible token |
| `currency` | Currency for price conversion (default: `usd`) |
| `chart_period` | Time period: `hour`, `day`, `week`, `month`, `year`, `max` |
