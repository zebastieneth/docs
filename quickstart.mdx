This guide walks you through four real API calls so you can see what Zerion returns and how to work with the data. You'll need an API key — grab one free at [developers.zerion.io](https://developers.zerion.io).

All examples use Vitalik's wallet (`0xd8dA6BF26964aF9D7eEd9e03E53415D37aA96045`) so you can copy-paste and run them immediately.

## 1. Get a wallet's portfolio

The portfolio endpoint gives you a high-level snapshot of a wallet: total value, position distribution, and daily changes.

<CodeGroup>

```bash cURL
curl -u "YOUR_API_KEY:" \
  "https://api.zerion.io/v1/wallets/0xd8dA6BF26964aF9D7eEd9e03E53415D37aA96045/portfolio"
```

```javascript JavaScript
const apiKey = "YOUR_API_KEY";
const address = "0xd8dA6BF26964aF9D7eEd9e03E53415D37aA96045";

const response = await fetch(
  `https://api.zerion.io/v1/wallets/${address}/portfolio`,
  { headers: { Authorization: `Basic ${btoa(`${apiKey}:`)}` } }
);

const { data } = await response.json();
console.log(`Total positions: ${data.attributes.total.positions}`);
```

```python Python
import requests

api_key = "YOUR_API_KEY"
address = "0xd8dA6BF26964aF9D7eEd9e03E53415D37aA96045"

response = requests.get(
    f"https://api.zerion.io/v1/wallets/{address}/portfolio",
    auth=(api_key, "")
)

data = response.json()["data"]
print(f"Total positions: {data['attributes']['total']['positions']}")
```

</CodeGroup>


## 2. List fungible positions

Now drill into the individual token positions. This returns every fungible token the wallet holds, with current values and quantities. Results are paginated — see [Pagination & Filtering](/pagination-and-filtering) for details.

<CodeGroup>

```bash cURL
curl -u "YOUR_API_KEY:" \
  "https://api.zerion.io/v1/wallets/0xd8dA6BF26964aF9D7eEd9e03E53415D37aA96045/positions/?filter[position_types]=wallet&sort=-value&currency=usd"
```

```javascript JavaScript
const apiKey = "YOUR_API_KEY";
const address = "0xd8dA6BF26964aF9D7eEd9e03E53415D37aA96045";

const response = await fetch(
  `https://api.zerion.io/v1/wallets/${address}/positions/?filter[position_types]=wallet&sort=-value&currency=usd`,
  { headers: { Authorization: `Basic ${btoa(`${apiKey}:`)}` } }
);

const { data } = await response.json();
data.forEach(pos => {
  const { name, quantity, value } = pos.attributes;
  console.log(`${name}: ${quantity.float} ($${value})`);
});
```

```python Python
import requests

api_key = "YOUR_API_KEY"
address = "0xd8dA6BF26964aF9D7eEd9e03E53415D37aA96045"

response = requests.get(
    f"https://api.zerion.io/v1/wallets/{address}/positions/",
    auth=(api_key, ""),
    params={
        "filter[position_types]": "wallet",
        "sort": "-value",
        "currency": "usd"
    }
)

for pos in response.json()["data"]:
    attrs = pos["attributes"]
    print(f"{attrs['name']}: {attrs['quantity']['float']} (${attrs['value']})")
```

</CodeGroup>

Key parameters used:
- `filter[position_types]=wallet` — only simple wallet holdings (not DeFi positions)
- `sort=-value` — highest value first
- `currency=usd` — values in USD (16 currencies supported)

## 3. Get parsed transactions

Zerion parses every transaction into human-readable operations. Instead of raw logs, you get structured actions like "Sent 1.5 ETH" or "Swapped USDC for ETH".

<CodeGroup>

```bash cURL
curl -u "YOUR_API_KEY:" \
  "https://api.zerion.io/v1/wallets/0xd8dA6BF26964aF9D7eEd9e03E53415D37aA96045/transactions/?filter[operation_types]=trade&filter[chain_ids]=ethereum&page[size]=3"
```

```javascript JavaScript
const apiKey = "YOUR_API_KEY";
const address = "0xd8dA6BF26964aF9D7eEd9e03E53415D37aA96045";

const params = new URLSearchParams({
  "filter[operation_types]": "trade",
  "filter[chain_ids]": "ethereum",
  "page[size]": "3"
});

const response = await fetch(
  `https://api.zerion.io/v1/wallets/${address}/transactions/?${params}`,
  { headers: { Authorization: `Basic ${btoa(`${apiKey}:`)}` } }
);

const { data } = await response.json();
data.forEach(tx => {
  console.log(`${tx.attributes.operation_type}: ${tx.attributes.mined_at}`);
});
```

```python Python
import requests

api_key = "YOUR_API_KEY"
address = "0xd8dA6BF26964aF9D7eEd9e03E53415D37aA96045"

response = requests.get(
    f"https://api.zerion.io/v1/wallets/{address}/transactions/",
    auth=(api_key, ""),
    params={
        "filter[operation_types]": "trade",
        "filter[chain_ids]": "ethereum",
        "page[size]": 3
    }
)

for tx in response.json()["data"]:
    attrs = tx["attributes"]
    print(f"{attrs['operation_type']}: {attrs['mined_at']}")
```

</CodeGroup>

Key parameters used:
- `filter[operation_types]=trade` — only swap/trade transactions
- `filter[chain_ids]=ethereum` — Ethereum mainnet only
- `page[size]=3` — limit to 3 results

Available operation types: `approve`, `burn`, `claim`, `delegate`, `deploy`, `deposit`, `execute`, `mint`, `receive`, `revoke`, `send`, `trade`, `withdraw`.

## 4. Subscribe to real-time webhooks

Instead of polling, you can receive instant notifications when transactions occur. Create a subscription to start receiving webhook callbacks at your URL.

<CodeGroup>

```bash cURL
curl -u "YOUR_API_KEY:" \
  -X POST "https://api.zerion.io/v1/tx-subscriptions/" \
  -H "Content-Type: application/json" \
  -d '{
    "data": {
      "type": "tx-subscriptions",
      "attributes": {
        "webhook_url": "https://your-app.com/webhooks/zerion",
        "chain_ids": ["ethereum", "base"],
        "addresses": ["0xd8dA6BF26964aF9D7eEd9e03E53415D37aA96045"]
      }
    }
  }'
```

```javascript JavaScript
const apiKey = "YOUR_API_KEY";

const response = await fetch(
  "https://api.zerion.io/v1/tx-subscriptions/",
  {
    method: "POST",
    headers: {
      Authorization: `Basic ${btoa(`${apiKey}:`)}`,
      "Content-Type": "application/json"
    },
    body: JSON.stringify({
      data: {
        type: "tx-subscriptions",
        attributes: {
          webhook_url: "https://your-app.com/webhooks/zerion",
          chain_ids: ["ethereum", "base"],
          addresses: ["0xd8dA6BF26964aF9D7eEd9e03E53415D37aA96045"]
        }
      }
    })
  }
);

const data = await response.json();
console.log(`Subscription created: ${data.data.id}`);
```

```python Python
import requests

api_key = "YOUR_API_KEY"

response = requests.post(
    "https://api.zerion.io/v1/tx-subscriptions/",
    auth=(api_key, ""),
    json={
        "data": {
            "type": "tx-subscriptions",
            "attributes": {
                "webhook_url": "https://your-app.com/webhooks/zerion",
                "chain_ids": ["ethereum", "base"],
                "addresses": ["0xd8dA6BF26964aF9D7eEd9e03E53415D37aA96045"]
            }
        }
    }
)

data = response.json()
print(f"Subscription created: {data['data']['id']}")
```

</CodeGroup>

Once created, Zerion will POST transaction data to your webhook URL whenever the watched wallets have activity on the specified chains.
