title: List All Tokens in a Wallet with USD Values
description: Fetch every fungible token a wallet holds along with current prices and USD values.

This recipe shows how to retrieve all fungible tokens held by a wallet, including their quantities, current prices, and total USD values.

## Prerequisites

- A Zerion API key ([get one here](https://developers.zerion.io))
- A wallet address to query

## Steps

<Steps>

### Fetch wallet positions

Call the positions endpoint with `currency=usd` and sort by value to get the most valuable tokens first.

<CodeGroup>

```bash cURL
curl -u "YOUR_API_KEY:" \
  "https://api.zerion.io/v1/wallets/0xd8dA6BF26964aF9D7eEd9e03E53415D37aA96045/positions/?currency=usd&sort=-value&filter[trash]=only_non_trash"
```

```javascript JavaScript
const apiKey = "YOUR_API_KEY";
const address = "0xd8dA6BF26964aF9D7eEd9e03E53415D37aA96045";
const encoded = btoa(`${apiKey}:`);

const response = await fetch(
  `https://api.zerion.io/v1/wallets/${address}/positions/?currency=usd&sort=-value&filter[trash]=only_non_trash`,
  { headers: { Authorization: `Basic ${encoded}` } }
);

const { data, links } = await response.json();

for (const position of data) {
  const { fungible_info, quantity, value, price, changes } = position.attributes;
  console.log({
    token: fungible_info.symbol,
    name: fungible_info.name,
    amount: quantity.float,
    price: price,
    value: value,
    change_24h: changes?.percent_1d,
  });
}
```

```python Python
import requests

api_key = "YOUR_API_KEY"
address = "0xd8dA6BF26964aF9D7eEd9e03E53415D37aA96045"

response = requests.get(
    f"https://api.zerion.io/v1/wallets/{address}/positions/",
    params={
        "currency": "usd",
        "sort": "-value",
        "filter[trash]": "only_non_trash",
    },
    auth=(api_key, ""),
)

data = response.json()

for position in data["data"]:
    attrs = position["attributes"]
    info = attrs["fungible_info"]
    print(
        f"{info['symbol']:>8} | "
        f"Amount: {attrs['quantity']['float']:>14.4f} | "
        f"Price: ${attrs.get('price', 0):>10.2f} | "
        f"Value: ${attrs.get('value', 0):>10.2f}"
    )
```

</CodeGroup>

Key parameters:
- `sort=-value` — highest value positions first
- `filter[trash]=only_non_trash` — excludes spam/dust tokens
- `currency=usd` — returns values in USD (also supports `eur`, `gbp`, etc.)

### Handle pagination for large wallets

If a wallet holds many tokens, the response will be paginated. Use the `links.next` URL to fetch subsequent pages.

<CodeGroup>

```javascript JavaScript
async function getAllPositions(address) {
  const allPositions = [];
  let url = `https://api.zerion.io/v1/wallets/${address}/positions/?currency=usd&sort=-value&filter[trash]=only_non_trash`;

  while (url) {
    const response = await fetch(url, {
      headers: { Authorization: `Basic ${encoded}` },
    });
    const { data, links } = await response.json();
    allPositions.push(...data);
    url = links.next || null;
  }

  return allPositions;
}
```

```python Python
def get_all_positions(address):
    all_positions = []
    url = f"https://api.zerion.io/v1/wallets/{address}/positions/"
    params = {
        "currency": "usd",
        "sort": "-value",
        "filter[trash]": "only_non_trash",
    }

    while url:
        response = requests.get(url, params=params, auth=(api_key, ""))
        result = response.json()
        all_positions.extend(result["data"])
        url = result["links"].get("next")
        params = {}  # params are included in the next URL

    return all_positions
```

</CodeGroup>

### Filter by specific tokens (optional)

To check balances for specific tokens, use `filter[fungible_ids]`.

```bash
curl -u "YOUR_API_KEY:" \
  "https://api.zerion.io/v1/wallets/0xd8dA6BF26964aF9D7eEd9e03E53415D37aA96045/positions/?currency=usd&filter[fungible_ids]=eth,0xa0b86991c6218b36c1d19d4a2e9eb0ce3606eb48"
```

This returns only ETH and USDC positions.

## Response fields reference

Each position object includes:

| Field | Description |
|---|---|
| `fungible_info.name` | Token name (e.g., "Ethereum") |
| `fungible_info.symbol` | Token symbol (e.g., "ETH") |
| `fungible_info.icon.url` | Token icon URL |
| `quantity.float` | Token amount as a decimal |
| `price` | Current price in requested currency |
| `value` | Total position value (quantity x price) |
| `changes.percent_1d` | 24h price change percentage |
| `relationships.chain` | Which blockchain the token is on |
