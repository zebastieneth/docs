## FAQ

<AccordionGroup>
  <Accordion title="How do I get an API key?">
    Visit [developers.zerion.io](https://developers.zerion.io) to sign up and get your API key.
  </Accordion>
  <Accordion title="What are the rate limits?">
    Rate limits depend on your plan. Check your dashboard or contact support for details.
  </Accordion>
  <Accordion title="Which blockchains are supported?">
    Zerion API supports multiple EVM chains (Ethereum, Polygon, Arbitrum, etc.) and Solana. Check the chains endpoint for the full list.
  </Accordion>
</AccordionGroup>
